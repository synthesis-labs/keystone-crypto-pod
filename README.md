## Installation

KeystoneCrypto is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following lines to your Podfile:

```ruby
source 'https://github.com/synthesis-labs/synthesis-specs.git' 
pod 'KeystoneCrypto'
```
